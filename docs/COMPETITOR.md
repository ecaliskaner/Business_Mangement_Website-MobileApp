# Insectram — competitor baseline

Researched Jul 2026. Source: vendor site + listings (see bottom).

## What they have

**Monitoring** — QR-coded monitoring points **and NFC stickers**; each trap has
a unique code technicians scan on inspection.

**Mobile app** — iOS/Android, offline capable. Workflow: view site info →
scan code → record activity → photograph non-conformities → select chemical
from database → capture signature → submit → auto report.

**Reporting** — ~10 report types (individual visit, trend, comparison,
analysis). Branded, audit-ready. Heat maps. Non-conformity tracking.
Automatic email delivery to client on completion.

**Compliance** — BRCGS, SALSA, ISO 22000, FSSC 22000, Red Tractor, AIB.
Audit packages auto-assembled by date / location / standard.

**Business** — job scheduling, GPS routing + route optimization, invoice
generation from visits or contracts, inventory management, CRM, client portal,
document storage, AI pest analytics.

**Commercial** — cloud, web + iOS + iPad/iPhone, from ~$23/month, free trial.
